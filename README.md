For training the network, the SWINySEG dataset and TJNU Cloud Detection Database (TCDD) were used. SWINySEG dataset has 6768 day and night time images of sky/cloud patches and their corresponding binary segmentation maps, while TCDD is composed of 1874 training images and 426 test images.
The SWINySEG dataset can be found at: http://vintage.winklerbros.net/swinyseg.html
The TCDD dataset can be found at: https://github.com/shuangliutjnu/TJNU-Cloud-Detection-Database
